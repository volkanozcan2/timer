# SAYAÇ

Çocukların sınav ve aktivitelerde zaman takibi yapması için geliştirilmiş, modern ve dayanıklı bir geri sayım aracıdır.

**Öne Çıkan Özellikler:**
*   **Çevrimdışı Destek:** İnternet bağlantısı kesilse bile çalışmaya devam eder. Yazı tipleri ve zaman senkronizasyonu için akıllı yedekleme (fallback) sistemleri içerir.
*   **Akıllı Zaman Senkronizasyonu:** İnternet varsa Türkiye saati ile otomatik senkronize olur, yoksa cihaz saatini kullanır.
*   **Görsel Atmosfer:** "Warp Tunnel" yıldız efekti ile odaklanmayı artıran bir arka plan sunar. Sol alttaki yıldıza tıklayarak kapatılabilir.
*   **Duyarlı Tasarım:** Ekran genişliğine göre otomatik boyutlanan devasa sayaç metni (%65 ekran genişliği).
*   **Saat/Sayaç Modu:** Sayaç metnine tıklayarak anlık saat ve geri sayım arasında geçiş yapabilirsiniz.
*   **Tam Ekran Modu:** Sağ alttaki ikona tıklayarak tam ekran deneyimine geçebilirsiniz.
*   **Sesli Uyarı:** Süre dolduğunda dikkat çekici bir alarm çalar.

**Teknik Detaylar:**
*   **Font:** Okunabilirlik için `Roboto Mono` kullanılır. Yüklenemezse sistem monospace fontlarına geçer.
*   **Altyapı:** Vite, Tailwind CSS ve Vanilla JavaScript ile geliştirilmiştir.

**Kurulum ve Çalıştırma:**

1.  Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```
2.  Geliştirme sunucusunu başlatın:
    ```bash
    npm run dev
    ```
